Telegram-bot
https://t.me/barbos13_bot

ищет вакансии на сайте  hh.ru по ключевым словам.
Вакансии для Москвы, размещенные текущей датой.

формат команды
/find КлючевоеСлово1 КлючевоеСлово2 КлючевоеСлово3
вместо '/find' можно набрать '/f'

Бот возвращает краткую информацию о вакансии
с самой большой рублевой заработной платой
и список всех вакансий в файле 'vac.txt'

Запускать надо модуль 'tele_bot.py'
Вспомогательные функции - в файле 'tele_hh_request.py'
Найденные вакансии хранятся также в 'vac_for_telega.json'

Реализованы также "функции вежливости":
приветствие / ответ на благодарность / прощание

Телеграмм стабильно работает без прокси.
По крайней мере, в Астраханской области,
где мы сейчас рыбачим...
